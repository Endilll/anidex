Upload torrent to anime trackers.

## Prerequisites

* NodeJS >= 7.8.0 (https://nodejs.org/)
* NPM >= 4.0.0 (https://www.npmjs.org/)

### Node Modules

After installing NodeJS with NPM goto `scripts` directory and type: `npm i`
